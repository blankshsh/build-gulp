build from gulp watch
===

Install
---

```
npm install
```

构造组件
---

- [gulp]
- [gulp-livereload]
- [gulp-postcss]
- [gulp-less]
- [gulp-watch]
- [gulp-concat]
- [gulp-uglify]
- [gulp-rename]
- [autoprefixer]
- [css-mqpacker]
- [csswring]


Usage
---

动态刷新需要sublime chrome安装LiveReload

```
gulp  // 调用
gulp watch  // 动态监控
```
